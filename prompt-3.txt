}

Bands (England/NI):
movinghome:
  0-125000 @0
  125001-250000 @2
  250001-925000 @5
  925001-1500000 @10
  1500001+ @12
additional:
  0-125000 @5
  125001-250000 @7
  250001-925000 @10
  925001-1500000 @15
  1500001+ @17
firsttime:
  0-300000 @0
  300001-500000 @5

Rule:
- If dutyType is firsttime and price > 500000 => use movinghome bands.

Calculation:
For each band:
  bandMax = price if max is null else min(max, price)
  taxableAmount = max(0, bandMax - min)
  tax = taxableAmount * rate/100
Label:
  if max null -> "Over £{min}"
  else -> "£{min} - £{max}"
totalTax sum, effectiveRate = totalTax/price*100
Store perBand only where taxableAmount>0 (or keep all if you prefer).
